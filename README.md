# vue-ripple-effect
a vue component with typescript
# install
```
npm i vue-ripple-effect
```
# use
``` js
import Vue from ''vue
import ripple from 'vue-ripple-effect'
import 'vue-ripple-effect/index.css'
Vue.use(ripple.install)
```
``` vue
<mb-ripple>
  <!-- slot -->
  content slot
</mb-ripple>
```
or set color
``` vue
<mb-ripple color="#fff">
  <!-- slot -->
  content slot
</mb-ripple>
```
